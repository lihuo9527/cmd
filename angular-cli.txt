
npm install -g cnpm --registry=https://registry.npm.taobao.org     装cnpm

cnpm install -g @angular/cli  创建ng-cli命令

npm install  

ng new demo3   创建新的项目

ng serve    运行ng项目   服务器IP  http://localhost:4200/

ng serve -p 3000   修改服务端口

ng g service 创建到指定的目录/需要创建的文件

ng g component my-new-component   创建一个完整的component

ng g directive my-new-directive   创建一个指令

ng g pipe my-new-pipe   创建一个管道

ng g service my-new-service   创建一个服务

ng g class my-new-class   创建一个class

ng g interface my-new-interface  创建一个接口

ng g enum my-new-enum   创建一个列举类别

ng build --prod 目录中就会出现dist文件夹，可以看到里面就是打包后的文件，包含一些html、js等文件，上传服务器就可以运行代码了。
